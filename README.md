## Description
Django feedback form with simple spam protection and customization possibilities.


## Installing
```bash
pip install -e git://github.com/eugena/django-mere-feedback.git#egg=django-mere-feedback
```


## Available Settings
Required:
```python
FEEDBACK_EMAIL
```

Optional:
```python
FEEDBACK_SUBJECTS
FEEDBACK_FORM_TEXT
FEEDBACK_FORM_SUCCESS_TEXT
```

Enjoy.
